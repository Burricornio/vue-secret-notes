# Proyect commands
cd vuex-auth-child-routes

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```

# Server commands
cd json-server

### Run json-server
```
json-server --watch db.json --middlewares login.js
```
